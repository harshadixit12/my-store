# 🛍️ Fake Store API

A minimal Node.js + Express-based fake store API.

---

## 📦 Features

- ✅ List products in store
- ✅ Filter products by ID
- ✅ Create an order
- ✅ List, and cancel orders

---

## 🚀 Getting Started

### 🧰 Requirements

- Node.js (v18+ recommended)
- npm

### 🔧 Setup & Run

```bash
npm install
npm start
