# 🛍️ Fake Store API

A minimal Node.js + Express-based fake store API.

---
Supports two resources -   
- Product
- Order

## 📦 Features

- ✅ List products in store `GET /products`
- ✅ Create an order `POST /orders`
- ✅ List orders `GET /orders`

---

## 🚀 Getting Started

### 🧰 Requirements

- Node.js (v18+ recommended)
- npm

### 🔧 Setup & Run

```bash
npm install
npm start
